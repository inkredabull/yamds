# yamds
Yet Another Music Discovery Service

## TECHS

- Bootstrap 4-alpha
- Bootflat
- Rails 4.2
- Heroku
- Soundcloud API
- bandsintown API
- Crowdflower API

## USAGE

`CF_TOKEN=<CF_TOKEN> CLIENT_ID=<CLIENT_ID> bundle exec rails s`

## BACKGROUND

This started as a two-day project for an inhouse Hackathon held at CrowdFlower Aug 20th/21st, 2015.
